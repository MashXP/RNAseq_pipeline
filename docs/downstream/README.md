# Downstream Pipeline: R Analysis

This directory contains the exhaustive dissection of the R scripts used for statistical analysis, functional biology, and visualization.

## Processing Flow

1.  **[01_bridge_data_prep.md](01_bridge_data_prep.md)**: Translates mentor counts into pipeline formats.
2.  **[01_data_prep.md](01_data_prep.md)**: Cleans and organizes sample-specific data.
3.  **[02_deseq2_dge.md](02_deseq2_dge.md)**: Runs the Differential Gene Expression engine.
4.  **[03_enrichment.md](03_enrichment.md)**: GSEA/GO/KEGG functional analysis.
5.  **04.x Visualization**: High-resolution plotting suite (PCA, Volcano, Heatmaps, etc.).
